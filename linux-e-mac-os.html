Linux e macOS

O UOL Messenger só está disponível para a plataforma Windows. Para os que usam outros sistemas operacionais, como Linux e macOS, sugerimos os seguintes comunicadores instantâneos multiprotocolos.

Linux

Para os usuários do Linux, recomendamos o comunicador multiprotocolo Pidgin. Gratuito, ele está disponível para diversas distribuições de Linux.

Visitar o site

macOS

Se você usa macOS (versão 10.2.7 ou superior) e quer instalar um comunicador instantâneo multiprotocolo, indicamos o Adium. Ele é gratuito e possui diversos extras para personalizá-lo.
